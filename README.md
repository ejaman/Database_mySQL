# dbcourse

## 소개
본 프로젝트는 MySQL과 php를 사용해 가상의 주민센터인 컬쳐랜드의 회원과 강사, 강의실 등을 관리하는 웹 개발입니다.
+ 사용자는 관리자로 로그인해 회원과 강사, 강의실을 관리할 수 있습니다.
